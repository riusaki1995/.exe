RCON

Asegúrate de habilitar el acceso a RCON en tu servidor, establecer una contraseña y ajustar la configuración correctamente. Cada perfil del panel puede tener su propia configuración guardada al exportar/importar o al usar perfiles guardados. La configuración predeterminada debería ser suficiente para la mayoría de los casos.

OBS

Puedes controlar OBS para cambiar escenas y fuentes automáticamente en ciertos eventos. Requiere OBS 28 o posterior. En OBS: Herramientas → Configuración del servidor WebSocket → habilita el servidor e indica el puerto (por defecto 4455) y la contraseña si la activas.

Streamer.bot

Streamer.bot es una de las herramientas más populares para los streamers. Podrás recibir eventos de Twitch y YouTube, además de ejecutar tus acciones desde Acciones y Eventos. Vincula tus cuentas de Twitch y YouTube en Streamer.bot y activa el servidor WebSocket en la pestaña Servers/Clients.

Acciones predeterminadas

Comandos listos para el servidor Java. Pulsa una tarjeta para añadirla a tu lista; luego configura el evento del directo con el lápiz en la fila.

Cómo se usa

  1. Construir comando: elige si escribes el comando a mano o lo generas eligiendo entidad, ítem o consumible (se rellena solo abajo).
  2. Revisa el comando; puedes editarlo siempre. El nombre en la lista se toma del comando o de lo que elijas en las listas. Luego sube una imagen y pulsa Guardar acción. El evento del directo (regalo, like…) lo configuras después con el lápiz en la fila.
Construir comando

Se genera un summon junto al jugador, con el mismo estilo que las acciones del catálogo (incluye {playername} en el nombre de la entidad).

Lista completa de ítems 1.21 + libros y equipos encantados (preset). Comando give @p … 1; los encantados usan sintaxis de componentes (servidor Java 1.20.5+). Más unidades: «Veces» o Comando libre.

Pociones genéricas sin efecto fijo; para pociones concretas usa Comando libre o edita el texto generado.

Veces y tiempos (por disparo)

Ej.: 3 veces y 1 s entre cada una: tres invocaciones o give separados. «Tiempo» retrasa todo el grupo (p. ej. 5 s antes de un solo zombie).

Comando que enviará RCON

Sin barra inicial. Puedes pegar un comando de consola de servidor Java o dejar el que se generó al elegir entidad/ítem.
